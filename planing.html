GIT y GITHUB

Objetivo general

creacion de un ova como diseño para una pagina de Git y GitHub.

obetivos especificos

  • 1. explicar a la sociedad la importancia de las herramientas y sus beneficios
  • 2. mostrar que es una plataforma muy fácil de manejar
  • 3. reducir las demoras en la búsqueda de herramientas eficientes y confiables

contenido

¿que es Git? Herramienta de control de versiones distribuida, de código abierto que los desarrolladores instalan localmente para gestionar el código fuente ¿que es GitHub? Plataforma basada en la nube, en línea para que los desarrolladores que utilizan Git puedan conectarse y cargar o descargar recursos
